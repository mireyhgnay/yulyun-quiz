## 📌 이 게임은?

- 조카를 덕질하는 마음에서 시작😊
- 조카의 재미있는 사실이나 이야기를 바탕으로 퀴즈 게임을 만듭니다.
- 가족 구성원이 퀴즈를 풀면서 조카에 대한 다양한 정보들을 알 수 있습니다.
- 가족끼리 모였을 때 가볍게 상품 걸고 게임으로 진행하기!
